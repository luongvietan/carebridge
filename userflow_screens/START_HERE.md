## Troubleshooting Quick Reference

### Login doesn't work
- Check email spelling exactly (copy-paste from credentials above)
- Check password is exactly `password123` (not TestPass123)
- Try an incognito/private browser window

### Can't find a page
- Check the URL in INSTRUCTIONS.md
- Look in the sidebar menu for the page name
- See "Alternative Routes" in INSTRUCTIONS.md

### Screenshots won't save
- Make sure you're saving to the correct folder
- Use PNG format
- Use the exact filename provided in INSTRUCTIONS.md

### Page looks wrong
- Hard refresh: Ctrl+Shift+R (Windows) or Cmd+Shift+R (Mac)
- Make sure zoom is at 100% (Ctrl+0)
- Try a different browser

For more help, see SCREENSHOT_CAPTURE_GUIDE.md
